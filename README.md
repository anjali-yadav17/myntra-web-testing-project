## 🎯 Project Overview & Objective

**Myntra** is one of India's leading e-commerce platforms specializing in fashion, lifestyle, and beauty. Fashion e-commerce introduces unique UX/functional complexities (e.g., dynamic size recommendations, multi-attribute product filters, secure checkout, complex coupon systems, and multi-size exchange cycles).

The primary objective of this project is to validate the core user journeys of Myntra to ensure a bug-free, seamless, secure, and responsive user experience.

---

## ⚙️ QA Methodologies & Types of Testing Performed

*   **Functional Testing**: Checked every feature (registration, search, cart, payments, checkout) against Myntra’s business requirements.
*   **Usability Testing**: Evaluated visual layouts, font legibility, responsiveness, and user friendliness of the "Size Chart" and "Fit Finder."
*   **Compatibility Testing**: Verified the website's responsiveness across different browsers (Chrome, Firefox, Microsoft Edge) and devices (Desktop, Laptop, Tablet, Mobile).
*   **Boundary Value Analysis (BVA) & Equivalence Partitioning (EP)**: Applied to input fields (e.g., promo codes, mobile numbers, PIN codes, product quantities).
*   **Security Testing**: Validated session management, secure HTTPS payment processing, password masking, and OTP expiration.
*   **Integration & End-to-End Testing**: Verified the seamless flow from product selection to payment gateway redirection and order creation.

## 🛠️ Testing Tools Simulating Industry Standards
